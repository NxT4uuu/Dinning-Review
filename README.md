# 🍽️ DiningReview Backend Application

A Spring Boot–based backend system for managing restaurant reviews and user interactions.  
This project provides APIs for users, restaurants, and admins to handle reviews, ratings, and dining experiences.

---

## ✨ Features

- **User Management**
  - Register and manage user accounts
  - Submit reviews for restaurants
  - Order food based on restaurant ratings and reviews

- **Restaurant Management**
  - Add and update restaurant details
  - View aggregated reviews and ratings
  - Search restaurants by name, location, or cuisine

- **Admin Management**
  - Moderate reviews (approve/reject)
  - Manage restaurant listings
  - Monitor user activity and review status

- **Review System**
  - Users can submit reviews with ratings
  - Admins can approve or reject reviews
  - Restaurants display average ratings based on approved reviews

---

## 🛠️ Tech Stack

- **Language**: Java  
- **Framework**: Spring Boot  
- **Database**: H2 (in-memory, with `data.sql` for sample data)  
- **Build Tool**: Gradle  
- **Persistence**: Spring Data JPA  
- **API Style**: RESTful endpoints
